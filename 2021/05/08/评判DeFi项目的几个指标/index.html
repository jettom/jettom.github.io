评判DeFi项目的几个指标

以下内容转自一个朋友的研究，文章结尾处有出处，一切内容仅用于提供信息，不应被视为法律，商业，投资或税务建议。

  1. 流通市值与完全稀释市值（FDV）
    流通市值代表着当前可以流通的代币的市值，完全稀释市值代表的是代币总量全部释放完以后的总市值。
    有很多项目刚开始由于流通量特别低，仅仅为总量的1%或者10%左右，这种项目会在随后的代币释放过程中产生比较大的卖压对币价的增长有比较大的影响。
    Curve刚刚发布CRV时，该代币的交易价格为$15-20以上，且该协议的FDV超过$50B。
    那比当时的以太坊还高！ 这个时候一旦买入了CRV就容易被埋。

  2. 总价值锁定（TVL）
    总价值锁定是DeFi常被大家使用的一个指标。它代表每个协议持有的资产总数，有人可以将其视为协议的资产管理（AUM）。
    一般来说，协议中锁定的值越多越好。
    这意味着人们实际上愿意将他们的资本锁定在协议中，在一定程度上信任它。也反应了该协议能带来更高的收入，提供更好的使用体验。
    但是TVL有两个很重要的点是
    a: TVL能不能保持稳定，并且长时间逐步上涨。
    b: TVL是在代币激励的情况下还是没有代币激励的情况下。例如Uniswap， aave以及yfi和mkr这几个的tvl都是在没有代币激励的情况下实现的。

  3. 协议收入
    协议的收入等于支付给协议的供应方参与者的总费用。对于AMM，这可能是支付给流动性提供者的总费用；
    对于借贷协议，这可能是借款人支付的利息金额。它实际上只是归结为用户愿意为使用该协议支付的费用。
    这就是为什么收入是一个至关重要的指标的原因-它表示着用户愿意使用协议的服务而支付的金额 。

  4. 协议收益
    收入是用户愿意支付给协议的金额，而该金额在很大程度上应归功于提供基础服务的供应方参与者，而协议收益是指代币实际产生的收益金额。
    就像早期的初创公司和成长型公司不向股东派发股息一样，并不是每个协议都将现金流分配给代币（因为当时可能不是对资本的有效利用）。
    例如尽管Uniswap在产生的收入方面显然是DeFi的领先者，但Uniswap的现金流量还没有累积到UNI代币持有者。相比之下，Sushiswap选择将交易手续费的约16％（0.30％手续费中的(1/6 0.05％)）分给xSUSHI持有者。

  5. 市销率（P / S）
    市销率（P / S）是将协议的市值与其收入进行比较。在传统金融中，市销率是衡量市场相对于资产产生的收益以及对未来增长的预期如何评估资产的基本指标。
    P / S比率的核心含义是市场愿意为每产生1美元的收入支付X美元。有趣的是，P / S比在不同协议里可能意味着不同的含义。这里有些例子：
    DEX：市场愿意为每赚取$ 1的交易费支付$ X
    贷款：市场愿意为借方支付的每1美元利息支付X美元
    收益：市场愿意为LP产生的每$ 1收益支付$ X
    因此，虽然P / S可能会引起细微差别，但它并不是衡量不同类型协议的最佳指标，但是，在比较同一类协议时，它是有价值的指标。

  6. 市盈率(P/E)
    由于大部分DeFi协议还处于早期发展阶段，通常并不为代币持有人分配收益。随着行业的成熟和更多协议将收益分配给代币持有者进行各种活动，市盈率将变得越来越重要。
    现在有一些协议（例如Maker，Sushiswap，Kyber等）可为代币持有者提供直接现金流。

  7. 费用/TVL比
    费用与价值锁定（TVL）比率可以用作了解协议从其持有的资本中产生费用的效率的指标。作为参考，我们通过将年度费用除以锁定值来计算该指标。
    与上述比率类似，这实际上是翻译为“协议能够为锁定的每个$ 1生成$ X”。
    “ x”越接近1，则该协议从底层资本赚取费用的效果就越有效，并且可能表明该协议比类似协议具有更好的投资。
